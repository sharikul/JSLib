JSLib is a new JavaScript library that wants to help you write normal JavaScript in normal JavaScript, with great functions here and there. The overall aim of JSLib is to make your code look more concise and readable.

Begin using JSLib

JSLib is just another JavaScript (.js) file, so it can be imported to your web pages via the script tag, like so: <script src="jslib.js"></script>
